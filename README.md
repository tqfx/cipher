# A cipher generator {#mainpage}

## prepare

Set the environment variable `VCPKG_INSTALLATION_ROOT` to point to [vcpkg](https://github.com/Microsoft/vcpkg), such as `VCPKG_INSTALLATION_ROOT=vcpkg`.

## release

```bash
cmake -S . -B build
cmake --build build
```
